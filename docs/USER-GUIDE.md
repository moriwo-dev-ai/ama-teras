# MyCodex 使い方ガイド(初心者向け)

MyCodexは、チャットで頼むとAIが**あなたのPCの中で実際に手を動かしてくれる**デスクトップアプリです。
ファイルを読み書きし、コマンドを実行し、アプリを作り、テストまで通します。
そして足りない機能があると、**自分で自分に新しい機能を作って組み込みます**(自己進化)。

---

## できること一覧

| カテゴリ | できること |
|---|---|
| コーディング | アプリ・スクリプト・テストの作成、バグ修正、リファクタリング(小〜中規模なら計画→実装→デバッグ→テストまで無介入で完遂) |
| ファイル操作 | 読む・書く・編集・検索(grep)・フォルダ整理。PC全体モードならプロジェクト外も(毎回あなたの承認つき) |
| コマンド実行 | npm・git など任意のシェルコマンド。devサーバのような常駐プロセスはバックグラウンド実行 |
| 計画的な長作業 | 計画ファイル(MYCODEX_PLAN.md)を自分で作って進捗管理。数十ターンの作業も迷子にならない |
| 並列作業 | サブエージェント最大3体に別々の作業を同時にさせる |
| 中断・再開 | アプリを閉じても(落ちても)会話が残る。再起動→セッション選択→「続けて」 |
| 巻き戻し | ファイル変更のたびに自動チェックポイント。失敗したらワンクリックで復元 |
| 自己進化 | 「◯◯できるツールが欲しい」→ 自分でツールを作り、テストし、あなたの承認を得て組み込む |
| スマホ操作 | iPhoneから進捗確認・指示・承認(Tailscale経由、外出先からもOK) |
| モデル選択 | Claude(既定: claude-fable-5)と OpenAI を設定で切替 |

## 最初のセットアップ(5分・初回だけ)

1. **起動**: `MyCodex Setup 0.1.0.exe` でインストールした場合はスタートメニューから。開発版は `npm run dev`
2. **APIキー登録**: Settings(⚙)→ APIキー欄に Anthropic(または OpenAI)のキーを貼って保存。
   キーはWindowsの暗号化機能で保存され、平文では残りません
3. **作業ディレクトリ**: Settings → 作業ディレクトリ で、AIに触らせたいフォルダを選ぶ。
   **ここが「AIの担当範囲」になります**(この外は原則触らない)
4. 以上。チャットに何か頼んでみてください:
   > 「このフォルダに何があるか教えて」

## 基本の使い方

### チャットで頼む

普通に日本語で頼むだけです。AIは必要に応じてツール(ファイル読み書き・コマンド実行)を使い、
その様子がチャット内にカードで表示されます。**「停止」ボタンでいつでも中断できます。**

### 承認ダイアログ — 一番大事な画面

AIが「何かを変える」操作をする前に、確認ダイアログが出ます:

- **色 = 危険度**: 読み取り(safe)/ファイル変更(write)/コマンド実行(exec)
- **diff表示**: ファイルを書き換える場合、何がどう変わるか差分で見える
- 選択肢は3つ: **許可**(1回だけ)/**セッション中許可**(この会話中は同じツールを聞かない)/**拒否**
- 面倒になったら Settings → 自動承認 で危険度ごとにOFFにできます
  (慣れるまでは write と exec は手動承認がおすすめ)

### 計画モード

送信欄の「plan」に切り替えると、AIは**実行せずに計画だけ**提示します。
内容が良ければ承認 → 通常モードで実行。大きめの依頼はまずplanで見るのが安全です。

## 長い作業を任せる(ここがMyCodexの本領)

### おまかせ設定(寝る前・外出前)

1. Settings → 自動承認 を全部ON
2. Settings → 最大ターン数 を 100 に(既定30。1〜200)
3. Settings → 編集後フック に `npm run -s test --if-present` を設定
   → ファイルを編集するたびにテストが自動実行され、失敗をAIが自分で見て直します
4. 頼み方のコツ: **「計画を立てて進めて」を添える** → MYCODEX_PLAN.md ができ、
   ヘッダ「計画」ボタンのパネルで進捗(n/m完了)が見えます

### 失敗したら巻き戻す

ファイル変更・コマンド実行のたびに自動でチェックポイントが取られています(gitの履歴は汚しません)。
Debugパネル → チェックポイント一覧 → 「復元」で、その時点のファイル状態に戻せます。
復元前の状態も自動退避されるので、行ったり来たりできます。

### 中断・再開

- アプリを閉じても、強制終了しても、会話はセッションとして自動保存されています
- 再起動 → チャット上部のドロップダウンからセッションを選ぶ → 「続きをやって」
- 「新規セッション」で新しい会話を開始(古いのはドロップダウンに残ります)

### 並列サブエージェント

> 「dispatch_agent を mode:"work" の parallel で2体使って、フロントとバックエンドを同時に進めて」

最大3体が同時に働きます。ヘッダ「エージェント」パネルで各自の状況が見え、
承認ダイアログには「🤖 サブエージェント #N」と出所が表示されます。
同じファイルを取り合うと2体目が拒否される安全装置つき。「停止」で全員止まります。

## 自己進化 — MyCodexの一番面白いところ

AIが「今の手持ちツールではできない」と判断すると(または、あなたが頼むと)、
**自分で新しいツールを作ります**。流れは全部見えます:

1. 進化パネルにジョブが現れる(別ブランチの隔離環境で作業するので、動いているアプリは無事)
2. 新ツールのコード+テストを自分で書き、検証ゲート(型チェック→テスト→実際に動かす)を通す
3. 全部合格すると**昇格承認ダイアログ**が出る — diffを見て、あなたが承認/却下
4. 承認するとアプリに組み込まれ、その場で使えるようになる。問題が起きたら自動で巻き戻し

頼み方の例: 「CSVをExcelに変換するツールが欲しい。request_capability を使って」

危険な要素(ネットワークアクセス・外部プロセス起動)が含まれる場合は、承認ダイアログに明示的な警告が出ます。

## PC全体モードと安全の仕組み

既定では、AIは**作業ディレクトリの中だけ**を触ります。
Settings → 操作範囲 を「fullPc」にすると、PC全体を対象にできます。ただし:

- **プロジェクト外の操作は、自動承認設定に関係なく毎回あなたに確認します**(「セッション中許可」も不可)
- 承認ダイアログに ⚠ プロジェクト外の警告バナーと、触ろうとしている実際のパスが表示されます
- Windowsシステム領域への書き込みや、アプリの認証情報(APIキー保存ファイル)へのアクセスは、
  確認すら出ずに**即拒否**されます
- プロジェクト外の操作はすべて監査ログ(audit.jsonl)に記録され、スマホの監査タブからも見られます

## スマホから使う

1. PC・iPhone 両方に Tailscale を入れて同じアカウントでログイン(初回だけ)
2. Settings → リモートアクセス を有効化 → 表示されたURLをiPhoneのSafariで開く
3. 共有メニュー →「ホーム画面に追加」でアプリ化(トークンは保存されるので次回から開くだけ)

できること: チャット送信・進捗のライブ表示・**承認**(スマホで許可するとPC側も閉じる)・
自己進化の昇格承認・監査ログ閲覧。外出先から「あの作業やっといて」→ 承認だけスマホで、が可能です。

接続URLのトークンは Settings → トークン再生成 でいつでも無効化できます。

## 設定リファレンス(Settings ⚙)

| 設定 | 意味 | おすすめ |
|---|---|---|
| プロバイダ / モデル | Anthropic か OpenAI か、使うモデル | claude-fable-5(既定) |
| APIキー | 各プロバイダのキー(暗号化保存) | — |
| 作業ディレクトリ | AIの担当フォルダ | プロジェクトごとに切替 |
| 自動承認(safe/write/exec) | 危険度ごとに確認を省略 | 慣れたらON、放置作業は全ON |
| 最大ターン数 | 1回の依頼でAIが動ける回数(1〜200) | 普段30、長作業は100 |
| 編集後フック | ファイル編集のたびに走るコマンド | `npm run -s test --if-present` |
| 操作範囲 | project(既定)/ fullPc | 必要な時だけfullPc |
| リモートアクセス | スマホ用サーバのON/OFF・トークン | 使う時だけONでも、常時ONでも |

## 困ったとき

- **途中で止まった(max_turns_reached)** → 最大ターン数を増やして「続きをやって」
- **アプリが落ちた/閉じてしまった** → 再起動してセッションを復元。「続きをやって」で再開
- **AIが変なことをした** → Debugパネル → チェックポイントから復元。会話で「◯◯を元に戻して」でも可
- **承認ダイアログが出ない・進まない** → スマホ側で承認待ちになっていないか確認(どちらからでも応答可)
- **スマホがつながらない** → iPhoneのTailscaleがONか → Windowsファイアウォールで「プライベートネットワーク」を許可
- **進化ジョブが失敗した** → 進化パネルのログとゲート結果を確認。失敗してもアプリ本体は無傷(隔離環境で作業しているため)
- **2つ起動してしまった** → 2つ目は自動で終了し、既存ウィンドウが前に出ます
