# M15 手動確認(3ペインUIリニューアル)

自動テスト(vitest 411件・typecheck 3構成)と、Claude Code によるCDPスモーク
(各ステップ計30項目超・既存機能の到達性込み)は 2026-07-06 に実施済み。以下は体感確認用。

## M15-1: レイアウト

- [ ] 左右ペインの境界バーをドラッグしてリサイズできる(再起動しても幅が保持される)
- [ ] ☰ / ◨(または Ctrl+B / Ctrl+J)でペインが折りたたみ/展開できる
- [ ] ウィンドウを900px未満に縮める → ペインがオーバーレイ表示になり、背景クリックで閉じる

## M15-2: 左ペイン

- [ ] プロジェクト(workspaceフォルダ)ごとにセッションがグルーピングされて見える
- [ ] **workspace が異なるセッションをクリック → workspace も自動で切り替わる**
      (右ペインの環境ウィジェットの📁表示が変わる)。エージェント実行中は切替不可のまま
- [ ] Ctrl+K で検索にフォーカス → 本文の単語(過去の会話に出た単語)でヒットする
- [ ] セッション右クリック → 名前変更(Enter確定・Esc取消)/ 削除
- [ ] 「+ 新規チャット」で現在のプロジェクトに新セッション、📁横の「+」でフォルダ選択→新プロジェクト

## M15-3: ファイルプレビュー

- [ ] チャットで「README.md を読んで」→ ツールカードの 📄 パスをクリック →
      右ペインに **レンダリングされたMarkdown** が表示される
- [ ] AIの返答本文中のファイルパス(点線下線)をクリック → コードが等幅表示される
- [ ] 画像ファイル(M14のスクショ成果物等)をプレビュー → 画像表示
- [ ] 「フォルダで表示」でエクスプローラが開く。「再読込」で編集後の内容が反映される
- [ ] workspace外のパスは project モードでは「fullPcでのみ表示できる」と断られる

## M15-4: 環境ウィジェットとタブ

- [ ] 右ペイン上部に 📁workspace / ⎇ブランチ / ±未コミット数 / checkpoint数 / ☑計画進捗
      が出る(git無しフォルダではgit行が消える)
- [ ] ヘッダの 計画/エージェント/進化/デバッグ ボタン → 右ペインの対応タブが開く
- [ ] 並列サブエージェント実行中、「エージェント」タブにバッジ(実行中数)が付く
- [ ] 進化ジョブ進行中、「進化」タブにバッジが付く
- [ ] 承認ダイアログは従来どおり画面中央のモーダル(タブに吸収されていない)

## M15.1: remote-ui バグ修正の再確認(2026-07-06追加)

- [ ] **スマホ写真添付**: remote-ui の📷から写真(数MBのカメラ撮影)を添付して送信 →
      PC側で処理され返信が返る(修正前は256KBのbody上限で413になり無反応だった。
      送信前に長辺1568pxへ自動圧縮される)
- [ ] **セッション切替**: remote-ui ヘッダの「セッション…」ドロップダウン → 過去の会話を選ぶ →
      履歴が入れ替わり、続きを送信できる(workspace も自動追従)。
      エージェント実行中はドロップダウンが無効化される
- [ ] **PWAトークン**:
  1. Settings → トークン再生成 → QRをiPhoneカメラで読む(URLに #t= が付いた状態で開く)
  2. **その画面のまま**共有 → ホーム画面に追加
  3. ホーム画面から起動 → トークン入力なしで接続される(追加時URLの #t= が引き継がれる)
  4. もしゲートが出ても1回入力すれば以後は保存される(説明文が画面に出ている)
- 注意: トークンを再生成したら旧QR・旧ホーム画面アプリは失効する(作り直し)

## 既知の制約(仕様)

- remote-ui(スマホ)は1カラムのまま(対象外)
- 本文中パスのリンク化はパターンマッチ(実在チェックはクリック時。無いファイルは
  プレビューにエラーが出るだけ)
- プレビューは読み取り専用(編集はAIに頼むか外部エディタ)
- ChatView上部のセッションドロップダウンは左ペインと重複するが互換のため残置
